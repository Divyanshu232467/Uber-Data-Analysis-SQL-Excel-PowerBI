# Uber Ride Data Analysis Dashboard
End-to-end Uber ride data analysis using SQL, Excel and Power BI. Includes data cleaning, SQL analysis, and interactive dashboard with business insights.

## Project Overview
This project presents an end-to-end analysis of Uber ride booking data using SQL, Excel, and Power BI.  
The objective is to analyze booking trends, revenue, cancellations, customer behavior, and driver performance through an interactive dashboard.

---

## Tools & Technologies Used
- **SQL** – Data Analysis  
- **Excel** – Data Cleaning & Preparation  
- **Power BI** – Data Visualization & Dashboard Creation  

---

## Dashboard Analysis
The Power BI dashboard includes the following insights:

- Ride Volume Over Time  
- Booking Status Breakdown  
- Total Bookings, Total Booking Value, Average Booking Value  
- Revenue by Payment Method  
- Cancelled Rides by Driver  
- Cancelled Rides by Customers  
- Top 5 Customers  
- Driver Ratings and Customer Ratings  
- Ride Distance Analysis  

---

## Key Business Insights
- Identified peak booking periods  
- Analyzed booking success and cancellation rate  
- Found major reasons for ride cancellations  
- Compared revenue generated by different payment methods  
- Identified top customers based on successful bookings  
- Analyzed driver and customer rating trends
